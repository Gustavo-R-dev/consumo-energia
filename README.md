# Calculadora de Consumo Elétrico Inteligente ⚡

### 🎯 Objetivo
Este sistema foi desenvolvido para ajudar o usuário a estimar o gasto mensal de energia de seus eletrodomésticos.

---

### 🛠️ Tecnologias
* **Linguagem:** Python
* **Plataforma:** GitHub

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![GitHub](https://img.shields.io/badge/github-%23121011.svg?style=for-the-badge&logo=github&logoColor=white)

---

### 📝 Fórmula utilizada
$ConsumoMensal = \frac{Potencia(W) \times HorasPorDia \times 30}{1000}$

O sistema realiza o cálculo financeiro baseado em uma tarifa média de R$ 0,80 por kWh:

$GastoMensal = ConsumoMensal \times 0,8$

---

### 🚀 Como executar
1. Certifique-se de ter o Python instalado.
2. Baixe o arquivo app.py.
3. No terminal, execute:
   ```bash
   python app.py
   ```
